# Lab System

مشروع Django لإدارة معمل التحاليل.

## المميزات الأساسية:
- إدارة المرضى
- إدارة التحاليل
- إدارة النتائج
- إدارة الفواتير

## طريقة التشغيل

1. تثبيت المتطلبات:
   ```bash
   pip install -r requirements.txt
   ```
2. تشغيل السيرفر:
   ```bash
   python manage.py runserver
   ```

## ملاحظات
- يمكن التوسع لاحقًا بواجهة React أو Electron.
- قاعدة البيانات الافتراضية SQLite ويمكن تغييرها لاحقًا.
